# Documentation utilisateur (version simplifiée)

## Présentation
SoundBook permet de rechercher des musiques, consulter des albums et gérer vos coups de cœur.

## Démarrer
1. Ouvrez l’application dans votre navigateur.
2. Créez un compte ou connectez‑vous.

## Fonctionnalités de base

### 1) Rechercher un son
- Utilisez la barre de recherche sur la page d’accueil.
- Saisissez un artiste, un titre ou un album.
- Les résultats apparaissent sous la barre de recherche.

### 2) Consulter un album
- Cliquez sur un résultat pour ouvrir la page de l’album.
- Vous y trouverez la liste des titres disponibles.

### 3) Aimer un titre (coups de cœur)
- Si vous êtes connecté, cliquez sur l’icône « J’aime ».
- Les titres aimés sont ajoutés à votre playlist « Coups de cœur ».

### 4) Accéder à votre profil
- Depuis la navigation, ouvrez votre profil.
- Vous pouvez y retrouver vos informations et votre bibliothèque.

## Remarques
- La connexion est nécessaire pour aimer des titres.
- Votre session est conservée sur votre navigateur.
